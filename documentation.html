Fetch Me

Documentation

Fetch Me is a website with different types of files you can fetch. These include: JSON, md and txt files.

How to fetch JSON file:


    fetch('https//fetch-me.netlify.app/resources/hello-world.json')
      .then(response => response.json())
      .then(response => {
        console.log(response["text"]);
      })
      .catch(error => {
        console.error(error);
      })

This will output:


Terminal

Hello, world!


How to fetch txt or md file:


    fetch('url of txt or md file')
      .then(response => response.text())
      .then(response => {
        console.log(response);
      })
      .catch(error => {
        console.error(error);
      })

This will output:


Terminal

Hello, world!

How to fetch (kinda?) png file:


    const myImg = document.createElement("img");
    myImg.src = "https://fetch-me.netlify.app/hello-world.png";
    document.body.appendChild(myImg);
